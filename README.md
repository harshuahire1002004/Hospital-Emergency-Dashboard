# 🏥 Hospital Emergency Dashboard

## 📌 Project Overview

This project focuses on analyzing hospital emergency room data using **Power BI** to improve operational efficiency and support data-driven decision-making.

The dashboard provides insights into patient visits, waiting times, satisfaction levels, and admission trends.

---

## 🎯 Objective

The main objective of this project is to create an interactive dashboard that helps hospitals:

* Monitor patient admission trends
* Track average waiting time
* Analyze patient demographics
* Evaluate department referrals
* Provide detailed patient-level insights
* Improve hospital service efficiency

---

## 🛠️ Tools Used

* **Microsoft Excel**

  * Data storage
  * Data cleaning
  * Dataset preparation

* **Power BI**

  * Data transformation (Power Query)
  * Data modeling
  * DAX calculations
  * Dashboard visualization
